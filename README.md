# Bootcamp Capstone
This is the repo for our team's TEKsystems Platform Engineer/Mainframe cohort capstone project. The objective of this project is to produce an automation solution that encompasses the variety of skills and technologies learned throughout the bootcamp.

# What it actually does
This project uses Ansible playbooks to add users (USER01, USER02, and USER03) to RACF and provision libraries in a way that developers would use in an industry environment. GitHub actions is used to check the validity of our code and maintain consistent formatting. To connect to z/OS, we are using a controlled environment provided by Innovation In Software.
